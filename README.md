# 🏥 Digital Patient Portfolio: Barangay Health Management System

A centralized web-based solution designed to modernize healthcare delivery in local Barangay health centers by replacing inefficient paper-based record-keeping with a secure, data-driven digital infrastructure.

## 🌟 Key Features
* **Automated Vaccine Tracking:** Real-time monitoring of pediatric immunization schedules.
* **Security Verification Layer:** Integrated confirmation protocols to prevent accidental clinical record updates.
* **Digital Patient Profiles:** Fast retrieval of patient history, vital signs, and UID-based portal linking.
* **Real-time Analytics:** Instant statistics for community health trend analysis.

## 🌍 Sustainable Development Goals (SDG) Alignment

### **SDG 3: Good Health and Well-being**
This system directly advances **SDG 3** by digitizing the immunization tracking process. It ensures that critical pediatric vaccines are administered on schedule through automated status monitoring and a secure verification system that minimizes human error in patient care.

### **SDG 11: Sustainable Cities and Communities**
The platform supports **SDG 11** by modernizing the essential infrastructure of the Barangay—the fundamental unit of local settlement. It replaces fragile paper archives with a resilient digital database, ensuring inclusive, organized, and reliable access to healthcare services for all community members.

## 🚀 Impact: Paper vs. Digital

| Feature | Paper-Based (Old) | Digital Portfolio (New) |
| :--- | :--- | :--- |
| **Data Retrieval** | Minutes of manual searching | < 3 seconds |
| **Data Integrity** | High risk of loss/physical damage | Secure database storage |
| **Analysis** | Nearly impossible | Longitudinal health trends |
| **Accuracy** | Prone to human clerical error | Verification-guarded updates |

## 🛠️ Technical Stack
* **Frontend:** HTML5, CSS3 (Google Fonts), JavaScript (SweetAlert2)
* **Backend:** PHP (Procedural/OOP)
* **Database:** MySQL
* **Styling:** Custom CSS with Outfit & DM Sans typography

## 💻 Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git)
    ```
2.  **Database Setup:**
    * Open XAMPP/WAMP and go to `phpMyAdmin`.
    * Create a new database named `barangay_health_db`.
    * Import the provided `barangay_health_db.sql` file.
3.  **Configure Connection:**
    * Ensure your database credentials in `db_conn.php` (or similar) match your local environment.
4.  **Run:**
    * Move the project folder to `htdocs`.
    * Open your browser and go to `http://localhost/your_folder_name`.
